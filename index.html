¿Que es HTML?

HTML son las siglas de HyperText Markup Language (Lenguaje de Marcado de Hipertexto). Es el lenguaje estándar utilizado para crear y estructurar el contenido de las páginas web.

Un documento HTML se compone de un conjunto de "elementos" grupados que definen tanto la información general del documento como el contenido visible para el usuario. Piensa en HTML como el "esqueleto" sobre el cual construyes toda una página web. No es un lenguaje que se encargue del diseño visual en sí (para eso está CSS), ni de la interactividad (JavaScript por acá), sino de organizar y estructurar la información. Sin embargo, ten presente que a día de hoy los 3 son comunmente usados para el desarrollo de una página web, siendo HTML el inicio de todo un recorrido para ser desarrollador FullStack.

Estructura HTML

Todo documento HTML comienza con un esqueleto básico compuesto por "elementos" para que el navegador los interprete de forma correcta, aquí vamos a desglozar una estructura básica de HTML:


1 <!DOCTYPE html>
2 <html lang="es">
3 <head>
4     <meta charset="UTF-8">
5     <meta name="viewport" content="width=device-width, initial-scale=1.0">
6     <title>Título de mi Página</title>
7     <!-- Enlace a CSS -->
8     <link rel="stylesheet" href="estilos.css">
9     <!-- Enlace a JavaScript -->
10     <script src="script.js" defer></script>
11 </head>
12 <body>
13     <!-- Aquí va el contenido visible -->
14 </body>
15 </html>

Linea 01. Etiqueta "<!DOCTYPE html>"

<!DOCTYPE html> es la declaración de tipo de documento para HTML. Es la primera línea que ves en casi todos los documentos HTML. Aunque parece una etiqueta HTML, técnicamente no lo es. Es una instrucción para el navegador web.

¿Para qué sirve?

Su propósito principal es decirle al navegador qué versión de HTML se está utilizando en el documento. Esto es crucial porque le indica al navegador cómo debe renderizar (mostrar) la página.

  • Modo Estándar (Standards Mode):

    • Cuando el navegador ve <!DOCTYPE html>, sabe que debe renderizar la página en el "modo estándar", lo que significa que seguirá estrictamente las especificaciones de HTML5 y CSS. Este es el modo moderno y el deseado.

  • Modo Quirks (Quirks Mode):

    • Si la declaración <!DOCTYPE html> está ausente o es incorrecta, el navegador podría entrar en "modo quirks". En este modo, el navegador intenta imitar el comportamiento de navegadores antiguos para renderizar páginas creadas antes de que los estándares web estuvieran bien definidos. Esto puede llevar a inconsistencias en el diseño y en la forma en que se ve tu página en diferentes navegadores, ya que el comportamiento del modo quirks varía.

3. Cómo Agregar Citas (Quotes)

HTML tiene etiquetas específicas para citar texto, mejorando la semántica de tu contenido:

Citas en Bloque (<blockquote>)

Para citas largas o bloques de texto que se extraen de otra fuente. Los navegadores suelen aplicar sangría.

La vida es lo que pasa mientras estás ocupado haciendo otros planes.

— John Lennon

<blockquote cite="URL_de_la_fuente">
    <p>Texto de la cita.</p>
    <footer>— <cite>Autor o Fuente</cite></footer>
</blockquote>

Citas en Línea (<q>)

Para citas cortas dentro de un párrafo o una oración. Los navegadores pueden añadir comillas automáticamente.

Como dijo la célebre frase, El conocimiento es poder.


<p>Como dijo la célebre frase, <q cite="URL_de_la_fuente">El conocimiento es poder</q>.</p>

Atribución de una Obra o Autor (<cite>)

Usa <cite> para el título de una obra creativa (libro, película) o el nombre de un autor, no para el texto de la cita en sí.

Mi libro favorito es Cien años de soledad.


<p>Mi libro favorito es <cite>Cien años de soledad</cite>.</p>

4. Cómo Agregar Sub-ítems a una Tabla HTML

Aunque no hay una etiqueta directa para "sub-ítems", puedes simularlos de estas maneras:

Opción 1: Usando `rowspan`

Cuando un ítem principal tiene varias características en columnas separadas.

Producto Característica Valor
Laptop X Procesador Intel Core i7
RAM 16 GB
Almacenamiento 512 GB SSD 

<td rowspan="3"><strong>Laptop X</strong></td>
<!-- ...las siguientes 2 filas ya no necesitan la celda del producto ... -->

Opción 2: Anidando Listas en una Celda

Si los "sub-ítems" son una lista de puntos o detalles dentro de una celda.

Fase Sub-tareas
Planificación
  • Reunión inicial
  • Análisis de requisitos
 

<td>
    <ul>
        <li>Elemento 1</li>
        <li>Elemento 2</li>
    </ul>
</td>

Opción 3: Usando Clases CSS para Estilos

Para jerarquías visuales simples con sangría y otros estilos.

Categoría Artículo
Electrónica Total Electrónica
Teléfonos móviles 

<tr class="main-item">...</tr>
<tr><td class="sub-item"></td><td class="sub-item">...</td></tr>

(Esta opción funciona mejor si luego le añades CSS para darle estilo a las clases `main-item` y `sub-item`).

5. Cómo Agregar Visualización de Código HTML

Para mostrar código en tus páginas:

Fragmentos de Código en Línea (<code>)

Para indicar que un pequeño fragmento de texto es código. El navegador le da una fuente monoespaciada.

La etiqueta para un enlace es <a>.


<p>La etiqueta para un enlace es <code>&lt;a&gt;</code>.</p>

Bloques de Código Preformateado (<pre>)

Mantiene los espacios, saltos de línea e indentación tal como los escribes. Ideal para bloques de código largos.

function saludar(nombre) {
    console.log("Hola, " + nombre + "!");
}
saludar("Mundo");
            

<pre>
function saludar(nombre) {
    console.log("Hola, " + nombre + "!");
}
saludar("Mundo");
</pre>

Combinación (<pre><code>)

La mejor práctica: <pre> mantiene el formato y <code> le da semántica de código. Recuerda usar entidades HTML para < (`&lt;`) y > (`&gt;`).


<pre><code class="language-html">
&lt;!DOCTYPE html&gt;
&lt;html&gt;
&lt;head&gt;
    &lt;title&gt;Ejemplo&lt;/title&gt;
&lt;/head&gt;
&lt;body&gt;
    &lt;h1&gt;Mi Página&lt;/h1&gt;
&lt;/body&gt;
&lt;/html&gt;
</code></pre>

Resaltado de Sintaxis (con JavaScript)

Para colores y temas como en los editores de código (ej. Highlight.js o Prism.js). **Esto solo funciona si enlazas librerías JavaScript externas**, no es HTML puro. Para este ejemplo, solo se muestra el contenedor sin la funcionalidad de resaltado.


// Ejemplo de JavaScript con resaltado de sintaxis
function calcularArea(radio) {
    return Math.PI * radio * radio;
}
const pi = Math.PI;
console.log(calcularArea(5));

6. Lo Imprescindible del HTML Básico (Resumen)

Para crear cualquier página web sencilla y bien estructurada, estos son los elementos y conceptos clave:

  • Estructura Fundamental: <!DOCTYPE html>, <html>, <head> (con `charset` y `viewport`), <title> y <body>.
  • Contenido de Texto: <h1>-<h6> (encabezados), <p> (párrafos), <strong> (importancia), <em> (énfasis).
  • Enlaces: <a href="...">.
  • Imágenes: <img src="..." alt="..."> (alt es vital).
  • Listas: <ul>/<ol> con <li>.
  • Etiquetas Semánticas (HTML5): <header>, <nav>, <main>, <section>, <article>, <aside>, <footer>.
  • Conceptos Clave: Etiquetas de apertura/cierre, atributos, etiquetas de cierre automático.
